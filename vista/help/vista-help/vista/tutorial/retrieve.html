Retrieving data by node:
    Click on the Node Menu. The first item in the menu would be a text field initalized to some node number. Edit this number to the node number for which you want the output displayed. Then select the appropriate item from the other menu items under the Node Menu list.

Retrieving data by arc:
    Click on the Arc Menu. The first item in this menu is a text field initalized to the previous selection. Enter the Arc number for which the output is desired and then click the appropriate item from the other menu items in the Arc Menu list.
