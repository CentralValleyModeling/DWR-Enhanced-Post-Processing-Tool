About a project

A project is an independent container, it contains every thing in the current frame, including all the input output dss files, related image, time window, mode, data view type settings, DTS and MTS, etc.

These are the operations you would operate on a project:

Create a new project:
There is a couple of ways calgui in the calsim root directory. If everything has been installed correctly, this should display a frame with a default blank project.

When you finish working to create a new project:

To start the gui, type on the current project, if you intend to create a new project, then save your current project and click on File | Project | New, a new blank project will be displayed.

Open an existing project:
Click on the File | Project | Open, a file dialog box will pop up, allow you to open the desired project.

You could also open an existing project through the command line when you start the gui by typing: calgui projectName.prj, a frame contains all the information about the project will be displayed.

Merge with an existing project:
Click on the File | Project | Merge With …, a file dialog box will pop up, allow you to load the desired project, and merge it with the current project.

Note: If there is anything conflict, the current project will overwrite the project be merged.

Save current project:
When you finished working on a project, always remember to save it. To save a project, click on the File | Project | Save/Save As, for Save As, a file dialog box will pop up, allow you to choose a project name and save it.

Exit:
To close and exit the gui, click on the left or right corner of the frame, or click on File | Exit. Always save your project before exit the system, otherwise, you might get a warning message.
