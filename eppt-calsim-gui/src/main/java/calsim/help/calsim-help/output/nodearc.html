Retrieving Node and Arc Data

Retrieving data by node:

Click the Node Menu, enter the node number, for which you want the output displayed, in the box, then select the type of data from the Node Menu list below the box.

Retrieving data by arc:

Click the Arc Menu, enter the arc number, for which you want the output displayed, in the box, then select the type of data from the Arc Menu list below the box.

Example1

Click on Node, change the node number to 6, click on Storage EOP. This displays the storage data of node S6 (Lake Oroville).

Example2

Click on Arc, change the node number to 9, click on Channel Flow. This displays the flow data of channel arc C9.
